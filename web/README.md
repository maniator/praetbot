# Next.js Bot Web Interface

This is the web interface for Praetbot, built with Next.js.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Available Pages

- `/` - Home page
- `/users` - Cookie leaderboard and user statistics
