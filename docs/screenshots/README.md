# Screenshots Directory

This directory contains screenshots of the Praetbot web interface and Discord bot in action.

## Current Screenshots

*Screenshots coming soon!*

To contribute screenshots:

1. Run the bot locally or in your deployment
2. Capture high-quality screenshots (PNG format preferred)
3. Redact any sensitive information (tokens, user IDs, etc.)
4. Add them to this directory
5. Update `../SCREENSHOTS.md` with references to your images

## Screenshot Guidelines

- **Resolution**: 1920x1080 or higher
- **Format**: PNG (lossless)
- **File size**: Optimize for web (< 1MB if possible)
- **Naming**: Descriptive kebab-case (e.g., `cookie-leaderboard-ui.png`)
- **Privacy**: Redact sensitive data, usernames, tokens
